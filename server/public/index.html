Siguiendo las tendencias tecnológicas actuales, quieren que todo el proyecto se base en JavaScript, por lo que quieren utilizar la “MEVN Stack” (del framework del front-end se encargará otro informático) y github para el control de versiones. La página debe mostrar información sobre las temperaturas que se están tomando en el CPD de la empresa, en tiempo real, así como poder filtrarlas para su mejor visualización. Se accederá a este servicio mediante un login. Cuando estas temperaturas sean anómalas, por otro lado, quieren recibir un aviso físico en un dispositivo remoto.     La toma de temperaturas, así como el streaming en tiempo real del CPD se consideran elementos de seguridad. Para cubrirlos todos a la vez, quieren utilizar un perro robot. Las piezas para el montaje del perro se imprimirán con una impresora 3D. El perro contará con una cámara y un sensor de temperatura. Obviamente, también tendrá que poder moverse de forma dirigida. La empresa quiere también documentar una evaluación de los riesgos más probables para sus trabajadores, así como una guía sobre cómo evitarlos.
